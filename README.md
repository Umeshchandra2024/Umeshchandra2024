## 💼 Research Project
**FedShield-CTI** — Federated Intrusion Detection via Heterogeneous CTI Graphs, Cryptographic Privacy, Verifiable Aggregation & Explainable Alerts
*Targeting ICDCN 2027 — 28th International Conference on Distributed Computing and Networking*
- Built a heterogeneous CTI graph pipeline converting raw NetFlow telemetry into typed threat graphs across 3 simulated orgs
- Implemented Bloom-filter PSI for privacy-preserving cross-org corroboration without disclosing topology
- Trained a federated edge-typed Relational GAT (FedProx) across orgs via Flower, lifting global F1 from 0.37 → 0.76
- Designed a verifiable on-chain audit layer (Solidity + Foundry/Anvil) committing per-round model hashes, with an HMAC signed-log fallback
- Coordinated a 5-person team across cybersecurity, AI/ML, and blockchain workstreams

## 🛠️ Featured Projects

### [HandyHive](https://github.com/Umeshchandra2024/Handyhive) — React.js, Node.js, Express.js, MongoDB, Cloudinary
A full-stack job marketplace connecting households with skilled workers for local services like repairs and home assistance. Features secure authentication, RESTful APIs for job posting and profile management, and Cloudinary-backed media storage.

### [KarunyaSetu](https://github.com/Umeshchandra2024/KarunyaSetu) — React.js, Tailwind CSS, Node.js, Solidity (Team of 4)
A decentralized disaster relief platform enabling transparent aid delivery via smart contracts, with zkProof-based privacy-preserving verification and Web3 wallet integration.

### [Plant Disease Classification](https://github.com/Umeshchandra2024/PlantDisease_classfication) — PyTorch, MobileNetV2, OpenCV, Grad-CAM
A deep learning image classification system that detects plant diseases from leaf images, using a custom CNN and transfer learning with MobileNetV2 on the PlantVillage dataset (20,638 images across 15 classes for Pepper, Potato, and Tomato). MobileNetV2 reached 92.99% test accuracy, with Grad-CAM used for model explainability by highlighting disease-affected regions influencing each prediction.
